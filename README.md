dotvim
======

Use vimod to manage vim plugins ( https://github.com/tpope/vim-pathogen.git is required ).

Usage:

  vimod <command> [<args>]

    commands:

      add <git url>
        Add a plugin from git

      ls
        List all installed plugins

      upgrade [all] | <plugin name>
        Upgrade all plugins or upgrade single plugin

      rm <plugin name>
        Remove a plugin
